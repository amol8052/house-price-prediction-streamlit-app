# House Price Prediction - Streamlit App

## 📌 Overview
This project predicts house prices using a Machine Learning model and provides an interactive UI using Streamlit.

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas / NumPy

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py
